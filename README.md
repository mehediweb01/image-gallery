# Image Gallery 
A modern and responsive image gallery application where users can upload, browse, search, filter, and preview images through a clean and intuitive interface. All features are fully functional and optimized for a seamless experience

---

## Project Overview  
This Image Gallery application is designed to provide a simple and enjoyable way to upload, organize, and explore images. Users can upload their own images, browse the gallery, search for specific content, filter images by category, and preview them in a larger view. The application features a modern, responsive interface that delivers a smooth experience across desktop, tablet, and mobile devices, with all core functionalities fully implemented and working reliably.

---

## Key Features  
- Upload images with ease.
- Search images by title.
- Filter images by category.
- Preview images in a larger view (modal).
- Fully responsive design for all devices.
- Fast loading and smooth user experience.
- Modern and intuitive user interface.
- All core functionalities are fully implemented and working seamlessly.

---

## Tech Stack  
Next.js · Tailwind CSS · TypeScript  

---

## Installation️ & Setup
1. Clone the repo and install dependencies:

```bash
git clone https://github.com/mehediweb01/image-gallery.git
cd image-gallery
npm install
```

2. Run the application:

```bash
npm run dev
```
